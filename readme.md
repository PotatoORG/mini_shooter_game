Window W H FL FS
FL is frame limit
FS determines whether the application is full screen (1) or not (0)

Font F S R G B
F - Font file
S - font sizw
r,g,b - font color

Player Specification
Player SR CR S FR FG FB OR OG OT V
SR - Shape Radius
CR - Collision Radius
S - Speed
FR, FG, FB - Fill color
OR, OG, OB - Outline color
OT - Outline Thickness
V Shape Vertices

Enemy Specification
Enemy SR CR SMIN SMAX OR OG OB OT VMIN VMAX L SI
SMIN minimum speed
SMAX maximu speed
VMIN minimum number of vertices
VMAX maximu number of vertices
L Small Lifespan
SI Spawn interval

Bullet Specification:
Bullet SR CR FR FG FB OR OG OB OT V L
SR - shape radius
CR - collision radius
FR, FG, FB - fill color
OR, OG, OB - Outline color
OT - Outline Thickness
V - shape vertices
L - lifespan
